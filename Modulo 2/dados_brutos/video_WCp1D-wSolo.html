About this video:
Calculus Concepts For Data Science|Calculus for Machine Learning|Statistics for Data Science
In this video, I explain basics of calculus concepts to be understood from data science point of view. I explain about constant slope vs changing slope in case of straight line vs curve. I explain concepts of derivative and integration with simple example as well so that it can be applied in the problem of data science and machine learning. This course offers a brief introduction to the multivariate calculus required to build many common machine learning techniques. I start at the very beginning with a refresher on the “rise over run” formulation of a slope, before converting this to the formal definition of the gradient of a function. We then start to build up a set of tools for making calculus easier and faster.
This video answers below question:
1. What is slope of a straight line
2.What is slope of a curve
3.What is significance of derivatives in machine learning
4.What is significance of integration in machine learning
5. How to use calculus for data science and machine learning

Publicado em 5 de mar. de 2020

Oracle Machine Learning (OML) comes bundled inside the Oracle Autonomous Database (ADB) and has been inside the Oracle Database for years. Oracle Machine Learning delivers a powerful converged, hybrid data management and machine learning platform that can more rapidly extract insights and find patterns in the data. Oracle Machine Learning leverages all the strengths of the Oracle Database(s) and can detect anomalies and noncompliant and potentially fraudulent expense reports, tax submissions, purchase, medical claims and other financial and operational data. OML’s library of 30+ in-database machine learning algorithms and AutoML help discover anomalous records and events that stand out in a multi-peculiar way. Using unsupervised and supervised ML algorithms, develop cloud and on-premises solutions to detect and combat fraud. In this session, see example scripts, OML notebooks and workflows, best practices and hear customer examples.

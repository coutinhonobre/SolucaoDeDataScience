I read an article the other which described how Airbnb uses computer vision and machine learning to automatically detect amenities (household objects) in their listings. I loved it so much I decided to replicate it in my own version as a machine learning project.

This video goes through manipulating the data (Open Images) I'm going to be using to train my machine learning model, Detectron2, the current state of the art for object detection.

One of the first steps in any machine learning project is preparing your data to be modelled.

In my case, this means manipulating Open Images images and labels to be in a format accepted by Detectron2.

Part 2 (data exploration) - https://youtu.be/hTHXVfryZDo

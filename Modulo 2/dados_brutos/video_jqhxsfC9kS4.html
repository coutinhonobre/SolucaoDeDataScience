AI is based on algorithms which allow input to become output. The Apriori algorithm that Eng Soon, Microsoft AI MVP, shares today is a categorization algorithm. Apriori machine learning algorithm works by identifying a particular characteristic of a data set and attempting to note how frequently that characteristic pops up throughout the set.

Listen to Eng Soon to learn about the basic overview of Apriori Machine Learning Algorithm, how it works. He will also show the references of the Apriori Algorithm using C#.

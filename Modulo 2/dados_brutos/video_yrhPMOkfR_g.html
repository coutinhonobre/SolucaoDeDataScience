Você tem interesse em se tornar um Cientista de Dados, não tem experiência e está com dúvidas sobre como começar e que passos seguir?

Também considera Data Science um assunto extremamente complexo e com muitos conceitos para aprender? Se sim, comente abaixo quais são as suas principais dúvidas que vou te ajudar!

Seja muito bem-vindo(a)! Este é o quarto de uma série de 100 vídeos nos quais vou mostrar os principais passos para que você tenha um guia sobre como começar e, quem sabe, se tornar um profissional diferenciado no mercado.

Neste vídeo vou falar dos Principais Objetivos no Uso de Ciência de Dados.
Há dois modos de utilizar os dados em Ciência de Dados: com o objetivo de explicar um comportamento que já ocorreu ou usar os dados para prever um comportamento futuro. Também falo das principais ferramentas utilizadas.

Na minha carreira em Consultoria, eu tive a oportunidade de lidar com diversos problemas de Ciência de Dados e com bastante prática eu acabei desenvolvendo um passo a passo para agilizar as análises. Foi com base nessa experiência que desenvolvi os vídeos para que você possa iniciar sua jornada. Embora eu tenha uma preferência pelo Python, você pode usar qualquer ferramenta.

Caso tenha interesse, em nosso site há um e-book gratuito com os principais conceitos para que você possa iniciar sua jornada! Espero que gostem.
